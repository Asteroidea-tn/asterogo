# AstroPkg - Asteroidea Go Utilities Library

A comprehensive collection of Go packages providing essential utilities for modern application development.

## 📦 Modules

Each module is located in its own folder with complete documentation:

- **`/encrypt`** - Encryption utilities  
  📄 See [encrypt/README.md](./encrypt/Readme.md) for details

- **`/env`** - Environment variable configuration loader  
  📄 See [env/README.md](./env/Readme.md) for details

## 📋 Project Structure

```
AstroPkg/
├── encrypt/          # Encryption utilities
├── env/              # Environment variable loader
├── go.mod            # Go module definition
├── Readme.md         # This file
└── notes.txt         # Development notes
```
